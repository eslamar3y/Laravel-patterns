Pattern Overviewنظرة عامة على الباترن

The Builder pattern separates the construction of a complex object from its representation using a fluent (method-chaining) interface.باترن Builder بيفصل عملية بناء الـ object المعقد عن تمثيله باستخدام fluent interface (method chaining).

It solves the "telescoping constructor" problem — a constructor with dozens of optional parameters that becomes impossible to read.بيحل مشكلة "telescoping constructor" — constructor بعشرات الـ parameters الـ optional اللي بيصعب قراءته.

Key Idea: Instead of a massive constructor, chain method calls: $report->format('pdf')->pages(3)->includeCharts()->build()الفكرة الأساسية: بدل constructor ضخم، اعمل chain للـ methods: $report->format('pdf')->pages(3)->includeCharts()->build()

Real Problem Exampleمثال على مشكلة حقيقية

You need to generate reports with many options: format, page range, header, footer, charts, watermark, language. A constructor can't handle all these elegantly.محتاج تعمل reports بخيارات كتير: format، نطاق الصفحات، header، footer، charts، watermark، لغة. الـ constructor مش قادر يتعامل مع كل ده بأناقة.

Laravel Implementationالتطبيق في Laravel

Laravel's Query Builder is the most famous Builder pattern in PHP. Chain methods to build a SQL query, only executing when you call get() or first().الـ Query Builder في Laravel هو أشهر تطبيق لـ Builder Pattern في PHP. بتعمل chain للـ methods لبناء SQL query، وبتتنفذ بس لما تستدعي get() أو first().

Laravel Code Exampleمثال كود Laravel

app/Reports/ReportBuilder.php
<?php
namespace App\Reports;
class ReportBuilder
{
    private string $format    = 'pdf';
    private string $title     = 'Report';
    private bool   $charts    = false;
    private int    $startPage = 1;
    private ?string $footer   = null;

    // Each method returns $this for chaining / كل method بترجع $this للـ chaining
    public function format(string $format): static
    {
        $this->format = $format;
        return $this;
    }
    public function title(string $title): static
    {
        $this->title = $title;
        return $this;
    }
    public function includeCharts(): static
    {
        $this->charts = true;
        return $this;
    }
    public function pages(int $from, int $to = 999): static
    {
        $this->startPage = $from;
        return $this;
    }
    public function footer(string $text): static
    {
        $this->footer = $text;
        return $this;
    }
    // Terminal method — produces the final product
    // الـ method الأخيرة — بتنتج الـ object النهائي
    public function build(): Report
    {
        return new Report($this->format, $this->title, $this->charts, $this->startPage, $this->footer);
    }
}
Usage / الاستخدام
<?php
// Custom Builder — clean, readable construction
// Builder مخصص — بناء واضح ومقروء
$report = (new ReportBuilder())
    ->title('Q4 Sales Report')
    ->format('pdf')
    ->includeCharts()
    ->pages(1, 10)
    ->footer('Confidential — Internal Use Only')
    ->build();

// Laravel's built-in Builder (Eloquent Query Builder)
// الـ Builder المدمج في Laravel — Query Builder
$users = User::query()
    ->where('active', true)
    ->where('role', 'admin')
    ->whereHas('subscriptions', fn($q) => $q->where('status', 'active'))
    ->orderBy('created_at', 'desc')
    ->paginate(15);

When to Use Itمتى تستخدمه

✓ Good Use Cases✓ حالات جيدة

  • Objects with many optional parameters
  • Constructing queries, documents, emails
  • Test factories with varied states
  • objects بـ parameters اختيارية كتير
  • بناء queries وdocuments وemails
  • Test factories بحالات متنوعة

✗ Avoid When✗ تجنبه عندما

  • Object only has 1-2 required parameters
  • PHP 8 named arguments already solve the problem
  • الـ object عنده parameter أو 2 فقط
  • Named arguments في PHP 8 بتحل المشكلة بالفعل

When NOT to Use Itمتى لا تستخدمه

PHP 8 named arguments reduce the need for Builder in some cases: new Report(format: 'pdf', title: 'Q4', charts: true) is already readable.الـ named arguments في PHP 8 بتقلل الحاجة للـ Builder في بعض الحالات: new Report(format: 'pdf', title: 'Q4', charts: true) مقروء أصلاً.
